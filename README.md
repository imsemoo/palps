# شبكة الصحافة الفلسطينية — ثيم HTML/CSS/JS

ثيم موقع أخبار ثابت (vanilla) جاهز للتركيب على أي لوحة CMS. عربي (RTL) + إنجليزي.

## البنية

```
index.html          الصفحة الرئيسية
article.html         مقال مفرد (عنوان + متن + اقتباسات + اقرأ أيضاً)
articles.html        أرشيف المعالجات (بحث + فلاتر + شبكة كروت)
author.html          صفحة الكاتب (نبذة + إحصائيات + مقالاته)
about.html           عن الشبكة (مانيفستو + أرقام + قيم + محطات)
contact.html         تواصل (فورم + قنوات + أسئلة شائعة)
assets/
  css/style.css      كل الاستايل لكل الصفحات (متغيرات + أنميشن + مكوّنات)
  js/main.js          كل التفاعلات فانيلا (مشتركة + خاصة بكل صفحة)
  img/logo.webp       اللوجو
_source/             ملفات التصميم الأصلية (DC) للرجوع — مش جزء من الثيم
README.md
```

كل الصفحات HTML دلالي بكلاسات فقط — **صفر استايل inline**. بيشاركوا نفس
`style.css` و `main.js`، ونفس الهيدر والتيكر والفوتر. الكلاسات الخاصة بكل
صفحة مسبوقة باسمها (`.article__`, `.articles__`, `.author__`, `.about__`,
`.contact__`) ومجمّعة في آخر `style.css`. تفاعلات كل صفحة بلوك IIFE مستقل
في آخر `main.js` يتحرّس بوجود عناصره (فبيعمل no-op على الصفحات التانية).

## التشغيل المحلي

افتح `index.html` في المتصفح مباشرة، أو شغّل سيرفر بسيط:

```
python -m http.server 8000   # ثم افتح http://localhost:8000
```

> الصور بليس‌هولدرز من picsum.photos والخطوط من Google Fonts (محتاجة إنترنت).

## التفاعلات (في `main.js`)

- **تبديل اللغة** عربي/إنجليزي + قلب اتجاه الصفحة (RTL/LTR).
- **الساعة اللحظية** في الهيدر ومركز الرصد.
- **المشغّل الصوتي**: تشغيل/إيقاف، تغيير السرعة، شريط التقدّم، الموجة، والمشغّل العائم.
- **مودال الفيديو**: يفتح بالضغط على أي فيديو، تبويبات مشاهدة/نص، ومقاطع ذات صلة.

## الربط بالـ CMS

النصوص والمحتوى دلوقتي بيانات تجريبية ثابتة. عند الربط:

1. **نصوص الواجهة** (اللي بتتبدّل مع اللغة) معلّمة بـ `data-i18n="key"` — موجودة في القاموس أعلى `main.js`.
2. **المحتوى التحريري** (العناوين، الأخبار، الصور) اقفز للأقسام المعلّمة بتعليقات `<!-- ... -->` في `index.html` وكرّر العنصر الواحد داخل حلقة الـ CMS بتاعتك.
3. الصور: بدّل روابط `picsum.photos` بحقل الصورة من الـ CMS.

## ملاحظة

كروت الأخبار في الصفحة الرئيسية لسه `<article>` من غير رابط — وقت الربط بالـ CMS
لُفّها في `<a href="article.html">` (أو رابط المقال الفعلي) عشان تبقى قابلة للضغط.
